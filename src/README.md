## 🚀 배포 방법

### 1️⃣ 코드 다운로드
Figma Make에서 **우측 상단 다운로드 버튼** 클릭

### 2️⃣ Vercel 배포 (무료)
```bash
# Vercel CLI 설치
npm install -g vercel

# 배포
vercel
```

### 3️⃣ 도메인 연결
queensmiracle.com → Vercel DNS 설정

**👉 자세한 내용은 `배포가이드.md` 참고**
